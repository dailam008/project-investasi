# Dashboard Investasi

Aplikasi Streamlit untuk memantau performa portofolio investasi dari Google Sheets secara real-time.

## Persiapan
1. Pastikan Anda memiliki Python yang sudah terinstal di sistem Anda.
2. Buat file `credentials.json` (Service Account dari Google Cloud Console) dan letakkan di folder utama proyek ini.

## Cara Menjalankan
1. Instal semua dependensi (disarankan menggunakan virtual environment):
   ```bash
   pip install -r requirements.txt
   ```
2. Jalankan aplikasi Streamlit:
   ```bash
   streamlit run app.py
   ```
